HTML, CSS, Git and Communication

This assignment is to demonstrate a basic understanding of how to structure an HTML document and override presentation styles using CSS. You will also become familiar with GitHub for version control and Slack as a communication tool.

  • Email Todd (me@toddwaits.org)
    • Provide your GitHub Username
    • Receive an invite link to the Slack Room.
  • Join the Slack Room using the link provided via email, and direct message @w8s (Todd Waits) once you arrive.
  • Create a repo in GitHub
    • Name the repo along the lines of: [username]-assignment-1. If you are concerned about a name, just message me.
  • Create an index.html file that introduces yourself, and apply CSS to override the default presentation of the HTML document in the browser.
    • I want to see you define a class and id attributes on HTML elements on your page.
    • If you are using a base CSS framework like Bootstrap or Foundation, I would like to you override the style of at least your defined class, ID, and a core HTML element.
    • See the Live Coding Example (Source)
